AutoHotkey Cookbook

A Howto cut-n-past guide for AutoHotkey

This Cookbook is meant to be a collection of short cut-n-paste examples. For those that have more time, we can include some larger examples. These will include personalised scripts and choices over which combinations of keys that will triggerw which function. Obviously some of these will collide with your own presumptions and or scripting practices. Stop Alt-F4
